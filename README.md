SpectralShield — Offline Web UI 🛡️🎵

This app runs locally — NO cloud, NO internet required.

📦 Requirements
- Docker Desktop: https://www.docker.com/products/docker-desktop/

🚀 How to Run

Windows:
  - Right-click run.ps1 → Run with PowerShell

Mac / Linux:
  - chmod +x run.sh
  - ./run.sh

✅ After launch:
Open your browser:
  http://localhost:8080

🛑 To stop the app:
docker compose down
